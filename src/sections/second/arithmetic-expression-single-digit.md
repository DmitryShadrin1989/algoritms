# Элементарное арифметическое выражение

## Условие

Дана строка с **однозначными** цифрами и операторами `+`, `-`, `*`, `/` **без пробелов**. Выражение **корректно**: два оператора не подряд, нет двузначных чисел (кроме как следствие промежуточных вычислений в коде — на входе только символы `0`–`9` по одной цифре на число-литерал).

Нужно вычислить **целочисленное** значение (для `/` — целочисленное деление, как в Java/Kotlin для `int`).

### Примеры

| Вход | Выход |
|------|------:|
| `2+2*3-4` | `4` |
| `-1+2*6/3-9/3` | `0` |

Проверка первого: `2 + (2*3) - 4 = 2 + 6 - 4 = 4`.

## Сложность

**O(N)** по времени и **O(N)** по памяти (стек термов), где `N` — длина строки.

## Замечания для интервью

- **Деление:** убедиться, что кандидат понимает: при целочисленном делении могут быть промежуточные «нецелые» в математическом смысле; если спросит — можно сказать, что **все деления в выражении дают целый результат без остатка** (как в примерах).
- **Последняя цифра:** обязательно **учесть** последний символ, если он цифра (типичный баг — забыть применить операцию к финальному числу).
- **Унарный минус** в начале (например `-1+...`): через искусственный **`+` в начале** и нулевой терм или отдельная обработка первого токена.

## Возможное решение (Kotlin)

Идея: один проход, **стек** «термов» для уровня `+`/`-`; при встрече `*`/`/` **склеиваем** с последним положенным в стек числом. Унарный `-` в начале обрабатывается за счёт стартового `op = '+'` и первого «оператора» `-`, который кладёт `0`, затем следующая цифра уйдёт с `-`.

```kotlin
import java.util.Stack

fun solution(expr: String): Int {
    var digit = 0
    var op = '+'
    val termStack = Stack<Int>()
    for (i in 0 until expr.length) {
        val c = expr[i]
        if (c.isDigit()) {
            digit = c - '0'
        }

        if (!c.isDigit() || i == expr.length - 1) {
            when (op) {
                '+' -> termStack.push(digit)
                '-' -> termStack.push(-digit)
                '*' -> termStack.push(termStack.pop() * digit)
                '/' -> termStack.push(termStack.pop() / digit)
            }
            digit = 0
            op = c
        }
    }
    var sum = 0
    while (termStack.isNotEmpty()) {
        sum += termStack.pop()
    }
    return sum
}
```

Для **Java** тот же алгоритм с `ArrayDeque<Integer>` как стеком (интерфейс `Deque` предпочтительнее устаревшего `Stack`).
