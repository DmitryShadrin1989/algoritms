## Условие

Дан массив целых чисел `x` длины `N`. Массив упорядочен по возрастанию. Нужно написать функцию, которая из этого массива получит массив квадратов чисел, упорядоченный по возрастанию.

### Пример

`[-3, 2, 4]` → `[4, 9, 16]`

Можно считать, что квадраты чисел помещаются в стандартный целочисленный тип (в тех языках, где это важно).

## Ожидания по решению

Хочется получить реализацию за **линейное время** с помощью **двух итераторов** (двух указателей).

Дополнительно можно обсудить конкретную реализацию:

- для **C++** — на алгоритмах STL (`transform`, `sort`) или позже на **C++20 ranges**;
- для **Python** — сделать `map` в виде генератора.

## Решение на Go

```go
func sortedSquares(nums []int) []int {
	result := make([]int, len(nums))

	l, r := 0, len(nums)-1
	for c := len(nums) - 1; l <= r; c-- {
		lsq, rsq := nums[l]*nums[l], nums[r]*nums[r]
		if lsq > rsq {
			result[c] = lsq
			l++
		} else {
			result[c] = rsq
			r--
		}
	}

	return result
}
```
