# First non-repeating character

## Условие (EN)

Given a string, find the **first** non-repeating character in it and return **its index**. If it does not exist, return **-1**.

## Примеры

| `s` | Результат | Комментарий |
|-----|-----------|-------------|
| `"yandex"` | `0` | `'y'` встречается один раз и стоит первым |
| `"yandex!ya!"` | `2` | первый уникальный символ — `'n'` на индексе `2` |

## Наивная идея (два прохода по строке)

1. Первый проход по `s`: в хеш-таблице накопить **частоты** символов.
2. Второй проход по `s` слева направо: вернуть индекс первого символа с частотой `1`.
3. Если такого нет — `-1`.

**Время:** `O(n)`, **память:** `O(k)`, `k` — число различных символов (для ASCII — константа).

## Альтернатива: один проход по строке + проход по map

Если строка **очень длинная**, второй **линейный** проход по `s` хочется избежать. Можно за **один** проход по `s` хранить для каждого символа пару `(count, last_index)` (или `(count, first_index)` — для уникальных символов важен **первый** индекс вхождения; см. код ниже).

После этого ответ — это **минимум индексов** среди символов с `count == 1`. Итерация идёт по **записям map** (их не больше, чем размер алфавита или число уникальных символов), а не по всей строке второй раз.

> Формально данные по строке всё равно нужно прочитать хотя бы **один раз** (`O(n)`). Оптимизация убирает **второй полный проход по символам строки**, заменяя его обходом map размера `O(k)`.

## Источник

LeetCode: [First Unique Character in a String](https://leetcode.com/problems/first-unique-character-in-a-string/)

## Решение на C++ (два прохода по `s`)

```cpp
class Solution {
public:
    int firstUniqChar(string s) {
        unordered_map<char, int> m;
        for (auto& c : s) {
            m[c]++;
        }
        for (int i = 0; i < s.size(); i++) {
            if (m[s[i]] == 1) {
                return i;
            }
        }
        return -1;
    }
};
```

## Решение на C++ (один проход по `s`, затем по map)

В цикле обновляется `second` как последний виденный индекс символа; для символов с `first == 1` это и есть единственное вхождение — его индекс. Для повторяющихся символов запись не участвует в минимуме.

```cpp
class Solution {
public:
    int firstUniqChar(string s) {
        unordered_map<char, pair<int, int>> m;
        int idx = s.size();
        for (int i = 0; i < s.size(); i++) {
            m[s[i]].first++;
            m[s[i]].second = i;
        }
        for (auto& p : m) {
            if (p.second.first == 1) {
                idx = min(idx, p.second.second);
            }
        }
        return idx == s.size() ? -1 : idx;
    }
};
```
