# Проверить симметричность бинарного дерева

## Условие

Дано бинарное дерево. Дерево **симметрично** (относительно вертикальной оси через корень), если левое поддерево **зеркально** совпадает с правым: у сопоставляемых узлов одинаковые значения, левый потомок слева соответствует правому потомку справа и наоборот.

### Структура узла (C++)

```cpp
struct Node {
    Node *l, *r;
    int value;
};
```

Нужно написать функцию, возвращающую `true`, если дерево симметрично, и `false` иначе.

## Идея решения

Рекурсивно сравнить два узла `a` и `b` как **зеркальные** позиции:

- оба `nullptr` → совпали;
- ровно один `nullptr` → не симметрично;
- оба не `nullptr` → значения равны, и рекурсивно `isMirrored(a->l, b->r)` и `isMirrored(a->r, b->l)`.

Корень: пустое дерево (`root == nullptr`) считают симметричным; иначе проверяют `isMirrored(root->l, root->r)`.

## Решение (C++)

```cpp
bool isSymmetric(Node* root) {
    return root == nullptr ? true : isMirrored(root->l, root->r);
}

bool isMirrored(Node* a, Node* b) {
    if (a == nullptr && b == nullptr) {
        return true;
    }
    if (a != nullptr && b != nullptr) {
        return a->value == b->value
            && isMirrored(a->l, b->r)
            && isMirrored(a->r, b->l);
    }
    return false;
}
```

В коде выше `NULL` заменён на `nullptr` в духе современного C++.

## Сложность

- **Время:** `O(n)` — каждый узел посещается константное число раз.
- **Память:** `O(h)` стек рекурсии, `h` — высота дерева (в худшем случае для вырожденного списка `O(n)`).

Итеративно можно обойти **двумя очередями** или одной очередью парами узлов — та же логика зеркала.
