# Счётчик максимального повторения символа

## Условие

Дана строка, например `"aafbaaaaffc"`. Для **каждого символа**, встречающегося в строке, нужно вывести **максимальную длину блока** из **подряд идущих** одинаковых символов (максимум среди всех непрерывных «пробегов» этого символа).

### Пример

**Вход:** `"aafbaaaaffc"`

**Выход** (формат по желанию):

```text
a: 4
b: 1
f: 2
c: 1
```

Интерпретация: для `a` самый длинный непрерывный фрагмент — `aaaa` (4 символа); для `f` — `ff` (2); `b` и `c` встречаются только одиночными блоками длины 1.

## Для кандидата

То же условие: по строке построить отображение **символ → максимальная длина его непрерывного повтора** в этой строке.

## Идея

Один проход слева направо: поддерживаем текущий символ `prev` и длину текущего пробега `count`. При смене символа сравниваем длину завершившегося пробега с уже сохранённым максимумом для `prev` и обновляем при необходимости. После цикла не забыть **последний** пробег.

**Время:** `O(n)` по длине строки. **Память:** `O(k)` для словаря, `k` — число различных символов.

## Пример решения (Python)

```python
def count_symbs(s):
    d = {}
    prev = ""
    count = 1
    for char in s:
        if char == prev:
            count += 1
        else:
            if prev and d.get(prev, 0) < count:
                d[prev] = count
            count = 1
        prev = char

    if prev and d.get(prev, 0) < count:
        d[prev] = count

    return d


assert count_symbs("aafbaaaaffc") == {"a": 4, "b": 1, "c": 1, "f": 2}
```

### Замечания

- **Пустая строка** — пустой словарь `{}`.
- Порядок ключей в словаре в примере `assert` фиксирован для читаемости; при сравнении в тестах удобнее сравнивать словари напрямую (в Python 3.7+ обычный `dict` сохраняет порядок вставки, но для проверки логики достаточно равенства множеств пар).
